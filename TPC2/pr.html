ABSTRACT:

Neste trabalho foi utilizada uma ferramenta Open-Source para defenir ontologias: o Protégé

Foi concebida uma ontologia descrevendo obras musicais e as suas partituras.

Para a adição de informação foi aplicada uma conversão de um ficheiro XML para a sintaxe Turtle.

Para persistência de dados foi utilizada uma base de dados orientada a grafos: o GRAPHDB

Finalmente, foram respodidas a algumas questões referentes aos dados, sendo estas respondidas através da linguagem de query SPARQL


Deliverables:

  • Dataset
  • Script de conversão
  • Ontologia
  • Querys(SPARQL)
